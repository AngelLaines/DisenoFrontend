Proyectos

Pagina de Soporte Tecnico

Pagina de reistro de computadoras con los datos de los alumnos que requieren de instalaciones de software especializado o mantenimiento en sus laptops. La pagina fue desarrollada en HTML con CSS, JavaScript y PHP.

Simulador de Tacos de Carne Asada

Un simulador de la preparacion de tacos de carne asada estimando el tiempo en el que se lleva el preparar uno o varios tacos de carne asada y los ingredientes que se requieren en total. EL simulador fue desarrollado solamente en Java utilizando hilos (Threads) e interfaz grafica.

E-commerce

Un E-commerce sobre la venta de ropa que permite publicar productos y calcular sus precios, asi como iniciar sesion y registrarse como usuario. El E-commerce fue desarrollado con HTML, CSS y Bootstrap, montando la pagina con Python Flask.

Roy Lavadoras

Un sistema de escritorio desarrollado para una tienda de refacciones y venta de lavadoras para guardar las ventas que realiza. El sistema fue desarrollado en C# y uso como base de datos MySQL.

Buscador de Frases

Una pagina que consiste en buscar frases de peliculas y guardarlas dependiendo del usuario. La pagina fue desarrollada en HTML con CSS y Python Flask.

Sistema Bancario

Un sistema bancario que consiste en 4 distintos sistemas que conformar el sistema de un banco, en el que se incluye gerencia, ejecutivo, ventanilla y cajero. Los proyectos fueron desarrollados en Visual Basic y como base de datos se tomo Access.
